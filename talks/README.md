# Talks

Notes on the recorded talks from AI Engineer 2026, built from the [AI Engineer YouTube
channel](https://www.youtube.com/@aidotengineer)'s videos and their transcripts. Every note
credits the speaker and links the video; timestamped links jump to the moments worth watching.
Summaries were written from auto-captions, which garble names and jargon, so spellings were
corrected against video metadata and obvious errors are hedged in the files.

Two kinds of coverage:

- [`live/`](live/) holds digests of the three full-day livestreams of the in-person keynote
  stages, segmented by talk with jump links.
- [`videos/`](videos/) holds one note per individual talk upload: 134 so far. Titles marked
  `*` were released on the channel around the event without a confirmed program placement
  (a few show internal evidence of being AIE Europe talks, noted in their files); unmarked
  titles are confirmed AI Engineer World's Fair Online Track 2026.

The channel is premiering the in-person World's Fair talks individually over the weeks after
the event. This folder covers everything with a transcript as of July 5, 2026, and the
pipeline that built it can be rerun as new uploads land.

## Highlights by theme

**The harness matters more than the model.** Etsy's HarnessBench measured a 24-point score
swing from the harness alone ([What If the Harness Mattered More Than the
Model](videos/what-if-the-harness-mattered-more-than-the-model.md)); Kaggle found harness
choice moves SWE-bench Pro 22% while frontier models sit within a couple points of each other
([Agentic Evaluations at Scale](videos/agentic-evaluations-at-scale-for-everybody.md)); Cline
climbed Terminal Bench largely by tuning containers and timeouts
([Evals Are Broken, Use Them Anyway](videos/evals-are-broken-use-them-anyway.md)); and
Microsoft's state-machine harness let Haiku replace Opus in a live voice tutor
([Don't Let the LLM Drive](videos/dont-let-the-llm-drive.md)).

**Verification is the product.** Sonar's benchmark leader still ships 614 bugs per million
generated lines ([Can LLMs Generate Enterprise-Quality
Code](videos/can-llms-generate-enterprise-quality-code.md)); Nebius caught coding agents
cheating benchmarks three ways ([SWE-rebench](videos/swe-rebench-lessons-from-evaluating-coding-agents.md));
hosted APIs are non-deterministic even at temperature zero, so record-and-replay beats rerun
([Your Agent Failed in Prod](videos/your-agent-failed-in-prod-good-luck-reproducing-it.md));
and a single successful replay is an anecdote, not a regression test
([Your Agents Need a Save Button](videos/your-agents-need-a-save-button.md)).

**Token economics decide architectures.** Tesco cut 94% of coding-agent input tokens with a
local code index ([We Cut 94% of AI Coding Tokens](videos/we-cut-94-of-ai-coding-tokens-with-local-code-index.md));
15 MCP servers can burn 100K tokens in definitions before the first message while skills cost
under 100 until activated ([Skills Are the New SDKs](videos/skills-are-the-new-sdks.md));
tool-selection accuracy collapses from 78% at 10 tools to 13.6% at 741
([The 100-Tool Agent Is a Trap](videos/the-100-tool-agent-is-a-trap.md)); and Phaidra held
100% correctness at flat token cost across 460K sensors by pairing an LLM planner with a
deterministic resolver ([Semantic Blindness](videos/semantic-blindness-500000-sensors-confused-an-llm.md)).

**Memory is more than a vector database.** Neo4j argued agents should retrieve decision
traces, not documents ([Decision Traces](videos/why-your-agents-need-decision-traces-not-just-documents.md));
a personal research OS made 10,994 notes agent-queryable with no vector database at all
([Turn 10,994 Notes Into Memory](videos/turn-10994-notes-into-memory.md)); and Turbopuffer's
ContextBench showed vector search cutting Claude Code's wasted file reads from one in three
to one in eight, while grep still wins import tracing
([Benchmarking Semantic Code Retrieval](videos/benchmarking-semantic-code-retrieval-on-claude-code.md)).

**Voice lives inside a latency budget.** The best deployable turn detector catches 58.9% of
finished sentences ([Voice Agents That Handle Interrupts](videos/voice-agents-that-handle-interrupts.md));
co-locating models with the orchestrator cut ~30% off response time
([Engineering Voice Agents](videos/engineering-voice-agents-latency-quality-and-scale.md));
and diarization error rates quintuple between headset and restaurant audio
([Beyond Transcription](videos/beyond-transcription-voice-ai-that-understands-conversations.md)).

**Security assumes the agent is compromised.** Tailscale's keyless sandbox makes revocation
un-routable-around ([What If the Network Was the Sandbox](videos/what-if-the-network-was-the-sandbox.md));
even the best models introduce vulnerabilities 20 to 40% of the time
([The AI Bugpocalypse](videos/the-ai-bugpocalypse-is-here-now-what.md)); and an SAE probe
detects sleeper-agent backdoors at 40x the signal of prior methods
([Your LLM Deception Monitor Is Broken](videos/your-llm-deception-monitor-is-broken.md)).

**The org chart is changing.** Block took 3,500 engineers to stage-five delegation and a 21x
lift in automated PRs ([Building an Autonomous Engineering
Org](videos/building-an-autonomous-engineering-org.md)); 500 Automattic employees started 794
projects in a 30-day vibe-coding experiment ([500 People Vibe
Coded](videos/500-people-vibe-coded-for-30-days-i-was-one-of-them.md)); and Accenture's
two-week build took twelve months to ship because five enterprise teams had to align
([Most Enterprise Agentic Projects Are Doomed](videos/most-enterprise-agentic-projects-are-doomed-heres-why.md)).
