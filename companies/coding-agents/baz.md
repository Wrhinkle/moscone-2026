# Baz

> Agentic code-review and planning platform: AI agents that read PRs (and now feature plans) to catch bugs, security issues, and convention violations before they ship.

- **Category:** coding-agents
- **AIEWF 2026 tier:** silver
- **Founded:** August 2023, Israel — Verified
- **Website / GitHub:** https://baz.co | https://github.com/baz-scm (public prompt library: `baz-scm/awesome-reviewers`)

## What they do
Baz runs specialized reviewer agents against every pull request — inside GitHub, no new tooling required — that learn a team's conventions/history and flag risky diffs, breaking API changes, and security issues, grouping large diffs into topics. Its newer **Planner** product moves this upstream: it routes a new feature/plan through loops that detect and pre-empt likely vulnerabilities before code is written, then rewrites the plan.

## Founders, funding & evidence of use
Founded by Guy Eisenkot (sold Bridgecrew to Palo Alto Networks for ~$200M, then VP Product there) and Nimrod Kor (also ex-Bridgecrew) — **Verified** (Calcalist). Raised $8M seed, extended to $17M total, co-led by Battery Ventures and boldstart ventures — **Verified**. Reports 100+ customers (Galileo, BigPanda, PayEm, Spotdraft) and ranks #1 on the (self-reported) precision-weighted Code Review Bench; partnerships include Amazon Bedrock and Datadog.
