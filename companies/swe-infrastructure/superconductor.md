# Superconductor

> A multiplayer cloud workspace where teams run and review parallel AI coding agents (Claude Code, Codex, Amp, OpenCode, Gemini) instead of one dev running one agent locally.

- **Category:** swe-infrastructure
- **AIEWF 2026 tier:** silver
- **Founded:** Not disclosed on-site; product live as of 2026 (Unverified precise year)
- **Website / GitHub:** https://www.superconductor.com · https://github.com/Superconductor

## What they do

Superconductor is built by **Volition, Inc.**, a product studio. It gives non-engineers (PMs, designers) and engineers a shared web/iOS/Slack surface to spin up isolated cloud sandboxes, launch multiple coding-agent backends on the same ticket, compare implementations side by side with live app previews, and merge via GitHub PRs — effectively an orchestration/QA layer on top of third-party coding agents rather than its own model.
