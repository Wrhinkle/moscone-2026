# Edge & Node

> The core software team behind The Graph protocol, now building verifiable/decentralized data infrastructure marketed toward AI agents as well as blockchain apps.

- **Category:** data-infrastructure
- **AIEWF 2026 tier:** supporting
- **Founded:** 2021, remote/distributed (Verified — Forbes launch coverage)
- **Website / GitHub:** https://www.edgeandnode.com/ ; The Graph protocol (thegraph.com); GitHub org graphprotocol

## What they do
Edge & Node is the primary development team maintaining The Graph — a decentralized indexing/query protocol that turns raw blockchain data into queryable GraphQL APIs ("subgraphs"). In engineer terms: instead of running your own chain-indexing infra, you query a subgraph endpoint and get structured on-chain data back. Edge & Node has more recently pitched this indexing/verification layer as infrastructure for autonomous AI agents that need verifiable, real-time on-chain and off-chain data (wallets, transaction simulation, multi-chain access) — a product line reportedly called "Amp" — positioning the same core tech (verifiable data provenance, decentralized indexing) for both institutional finance and agentic AI use cases.

## Founders & origins
Founded by Jannis Pohlmann and Tegan Kline as the core team spun out to build The Graph (Verified — Forbes, 2021). Tegan Kline later became CEO (Reported — The Block). Current CEO as of 2025 reported as Rodrigo Coelho (Reported — DL News).

## Funding
The Graph protocol raised ~$60M total across 7 rounds from investors including Multicoin Capital, Coinbase Ventures, and Tiger Global (Verified — Crunchbase/Tracxn). A related $205M ecosystem fund was announced in 2022, though that is a grants pool, not a direct equity investment (Reported).

## Evidence of real-world use
The Graph/subgraphs are widely used across the Ethereum/web3 developer ecosystem for dapp data queries (large GitHub footprint under graphprotocol, long-running mainnet protocol since 2020). The AI-agent pivot ("Amp") is newer and less independently verified — mostly covered via DL News interview with Edge & Node's CEO rather than named enterprise case studies.

## Relevance to agentic AI engineering
Relevant to agent tool-use around verifiable data access and on-chain agent wallets/transactions — an emerging niche (agentic crypto/agent-to-agent payments) rather than mainstream LLM-agent tooling. Worth evaluating if building agents that need trust-minimized data provenance or on-chain state.
