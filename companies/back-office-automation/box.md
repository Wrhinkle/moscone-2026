# Box

> Public cloud content-management company (NYSE: BOX) building permission-aware AI agents over enterprise files.

- **Category:** back-office-automation
- **AIEWF 2026 tier:** silver
- **Founded:** 2005, Redwood City, CA — Verified
- **Website / GitHub:** https://www.box.com | https://developer.box.com

## What they do
Box is the established enterprise content/file-management platform (public since 2015) that has layered on an AI stack: **Box AI Studio**, a no-code builder for custom agents scoped to up to 100 files or a "Hub"; the **Box Agent**, a reasoning agent that searches, synthesizes, and generates content across a company's files while respecting existing permissions/governance; and a **Box MCP server** (hosted or self-hosted) exposing enterprise content securely to Claude, Microsoft Copilot Studio, Azure API Center, and Mistral Le Chat (Salesforce Agentforce and GitHub Copilot listed as coming soon).

## Founders, funding & evidence of use
Founded by Aaron Levie and Dylan Smith (with Jeff Queisser and Sam Ghods), seeded by Mark Cuban ($350K, 2005) — **Verified**. Raised ~$693M pre-IPO; public company since 2015. Real-world use is strong by default (large existing enterprise customer base pre-AI); AI-specific evidence is CEO-driven press (TechCrunch, GeekWire) rather than third-party case studies at this point.

## Relevance to agentic AI engineering
Directly relevant to back-office automation: permission-aware retrieval/agents over unstructured enterprise content is exactly the "job packet"/document-grounded agent pattern this repo tracks.
