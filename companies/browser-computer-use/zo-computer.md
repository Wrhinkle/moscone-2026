# Zo Computer

> A personal cloud computer — a Linux server plus an AI agent with filesystem access, code execution, scheduling, and hosting — reachable by chat, text, or email.

- **Category:** browser-computer-use
- **AIEWF 2026 tier:** supporting
- **Founded:** 2023, Brooklyn, NY (Reported — Tracxn)
- **Website / GitHub:** https://www.zo.computer

## What they do
Zo gives each user a persistent personal cloud server (100GB storage) paired with an AI agent that can read/write files, execute code, run scheduled/background jobs, and host services — effectively "computer use" as a durable environment rather than a one-off browser session. Unlike a chat window, state persists: files and projects live on the server continuously, and the AI keeps running even after you close the browser. Users can email or text commands to their Zo instance, turning it into an always-on personal assistant/dev box. It bundles access to multiple frontier models for chat, image/video generation, and transcription. Free tier available; paid credits unlock more model access — a real (if early) commercial product, not just a demo.

## Founders & funding
Cofounded by Ben Guo (early engineer at Stripe) and a cofounder referred to as Rob (reported as an early Substack engineer) (Reported, single-source LinkedIn/newsletter bios). Backing reportedly includes Craft Ventures, Factorial Capital, and Lightspeed Venture Partners, with informal investor mentions of the Collison brothers (Stripe) and Guillermo Rauch (Vercel); total raised ~$7.8M (Reported — Tracxn/f4.fund, not independently cross-verified against a primary funding announcement).

## Relevance to agentic AI engineering
Sits squarely in the browser/computer-use agent category — a persistent, stateful "agent desktop" complementary to Browserbase/CUA-style ephemeral browser automation. Relevant to this repo's agent-memory and long-running-agent-infrastructure research areas.
